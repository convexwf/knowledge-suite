# knowledge-suite Journal - 2023-11-01

## Batch 1: Side panel Save dropdown + inline batch workflow

Implemented the first batch of the batch UX redesign:

- **Split Save button**: Save button now has a dropdown arrow. Clicking Save saves the current page (unchanged). Clicking the arrow reveals "Save Section" option.
- **Removed Batch tab**: The Batch tab has been removed from the diagnostics area. No independent Batch tab exists anymore.
- **Inline batch workflow**: Instead of switching to a Batch tab, the batch confirmation panel, progress, and results are rendered inline by replacing the Preview area content.
- **Progress bar**: Added a `<progress>` element for visual progress during batch jobs.
- **Completion actions**: After batch completion, "Open in Items" and "Back to Preview" buttons are shown.
- **Removed "Discover Section" button**: Removed from the More menu since it's now accessible via the Save dropdown.
- **Save dropdown auto-close**: Clicking outside the dropdown closes it.

CSS additions: `.save-button-group`, `.save-dropdown`, `.save-dropdown-menu`, `.batch-inline` styles.
